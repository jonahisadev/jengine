JEngine

A cross-platform 2D C++ game engine

GitHub | Latest Release | Documentation | Report a Bug

Features

  • Cross-platform (tested on Windows, Mac, and Linux)
  • Easy to get up and running (minimal startup code)
  • Basic quad/texture rendering
  • Modern OpenGL (3.3)
  • Spritesheets
  • Font loading and rendering (TTF)
  • WAV loading and playback (+Lowpass filter)
  • Basic animation

Open Source Libraries Used

  • SOIL - Image loading
  • GLEW - Windows OpenGL implementation
  • GLFW - Cross-platform Window/OpenGL context
  • FreeType - Font glyph loading
  • OpenAL Soft - Audio playback
  • GLM - OpenGL matrix/vector/transform library
